Welcome to Procurement Internet Net Promoter Score (iNPS)!



INPS tool is used to send surveys to the requestors. Data comes to this tool from BDW PO, BDW Engagement support, POP, IBM emptoris data in excel file. Data from BDW PO, BDW engagement support, POP tool will be pulled by scheduled agents in INPS tool. IBM Emptoris data in excel file will be imported to INPS tool. IBM users take the survey with the Adhoc link. If the users submit the feedback as dissatisfied notification will be sent to the corresponding Geo leader stating that the user is dissatisfied with the support given, Geo leader or focal have to work with the requestor and make sure that the requestor is satisfied with the support provided and close the feedback request.
